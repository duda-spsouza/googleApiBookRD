Projeto de integração com a api books  fazendo o uso de tecnologias como HTML5, Angular e o Bootstrap
Diretório
app 

    services.js = integração com a api
    filters.js = são os filtros que foi utilizado no projeto como  autor, data no formato brasileiro e título;
    controller.js = para listagem dos livros, localstorage, eventos do modal de detalhes do livro
    app.js = estão as dependências e a rota para o html de listagem dos livros
